1. creo db
2. installo pacchetti necessari
3. composer install
4. copio .envexample e lo rinominop .env
5. modifico il file env con la connessione al database
6. pulisco la cache
    php artisan config:clear
7. genero la key
    php artisan key:generate
8. npm install
9. controllo installazione
    php artisan serve - npm run watch
10. lancio le migration
11. effettuo la registrazione al sito
12. ok buon lavoro
